# Навигационный гайд по Pharmo CRM

## Маршруты

### Авторизация
- `/login` — вход по логину и паролю.
- `/register` — регистрация нового аккаунта.
- `/forgot-password` — запрос ссылки для сброса пароля.
- `/reset-password` — установка нового пароля по ссылке.
- `/verify-email` — подтверждение email.

### Основное
- `/dashboard` — главная страница: приветствие и тепловая карта визитов.
- `/activs` — список визитов с поиском, фильтром по статусу и пагинацией.
- `/activs/create` — форма создания нового визита.
- `/activs/{id}` — карточка визита со статусом, временем, препаратами и действиями.
- `/activs/{id}/edit` — редактирование визита.
- `/calendar` — календарь визитов в режимах «Месяц» и «Неделя».
- `/orgs` — список организаций с поиском.
- `/orgs/create` — добавление организации, доступно только администратору.
- `/orgs/{id}` — карточка организации.
- `/orgs/{id}/edit` — редактирование организации, доступно только администратору.
- `/physes` — список врачей с поиском.
- `/physes/create` — добавление врача, доступно только администратору.
- `/physes/{id}` — карточка врача.
- `/physes/{id}/edit` — редактирование врача, доступно только администратору.
- `/map` — карта организаций по координатам.
- `/map/track` — трекинг точек закрытых визитов на карте за период.

### Аналитика и отчеты
- `/analytics` — графики и сводки по визитам. Видно менеджеру, директору и администратору.
- `/reports` — таблица визитов за период с экспортом в CSV и Excel. Видно менеджеру, директору и администратору.

### Администрирование
- `/admin` — панель администратора с вкладками «Пользователи», «Департаменты», «Препараты», «Специальности».
- `/admin/audit-log` — журнал изменений сущностей.
- `/profile` — личный кабинет: имя, фамилия, телефон, смена пароля, тема и выход.

## Навигация

Слева находится sidebar. На мобильных он открывается кнопкой меню в левом верхнем углу. Sidebar разбит на группы:

- «Основное»: «Главная», «Визиты», «Календарь», «Организации», «Карта», «Трекинг», «Врачи».
- «Аналитика»: «Аналитика», «Отчеты». Группа видна менеджеру, директору и администратору.
- «Система»: «Администрирование». Группа видна только администратору.

Внизу sidebar находится карточка пользователя с именем и ролью. Клик по ней открывает `/profile`. Там же находятся переключатель темы и кнопка выхода. Кнопка сворачивания sidebar находится у правого края панели.

На страницах списков «Визиты», «Организации» и «Врачи» кнопка основного действия находится в правом верхнем углу. Для визитов это «Новый визит», для организаций и врачей — «Добавить». Добавление организаций и врачей доступно только администратору.

Раздел «Препараты» отдельной страницы для обычного пользователя не имеет. Препараты можно найти в форме создания или редактирования визита. Администратор управляет препаратами в `/admin`, вкладка «Препараты».

## Создать визит

1. Открыть в sidebar раздел «Визиты».
2. Нажать «Новый визит» в правом верхнем углу.
3. В блоке «Цель визита» выбрать вкладку «Организация» или «Врач».
4. Через поле поиска выбрать конкретную организацию или врача.
5. Указать «Дата начала».
6. При необходимости заполнить «Описание».
7. В блоке «Препараты» добавить препараты через выпадающий поиск.
8. Нажать «Создать визит». После создания откроется карточка нового визита.

## Найти препарат

Обычный пользователь ищет препарат в форме создания или редактирования визита, в блоке «Препараты». Нужно кликнуть по полю поиска и начать вводить название препарата.

Администратор может искать и редактировать препараты в `/admin`, вкладка «Препараты».

## Найти врача

1. Открыть в sidebar раздел «Врачи».
2. Ввести в поиск ФИО, специальность или контактные данные.
3. Нажать на строку врача, чтобы открыть карточку.

## Найти организацию

1. Открыть в sidebar раздел «Организации».
2. Ввести в поиск название, ИНН или адрес.
3. Нажать на строку организации, чтобы открыть карточку.
4. Альтернативный путь — раздел «Карта»: там есть поиск по имени и адресу, а также фильтр по типу организации.

## Открыть визит и поменять статус

1. Открыть «Визиты» или «Календарь».
2. Нажать на нужный визит.
3. В карточке визита доступна цепочка статусов: «Запланирован» → «Открыт» → «Сохранен» → «Закрыт». Также возможен статус «Отменен».
4. Если визит запланирован, доступна кнопка «Открыть визит».
5. Если визит открыт, доступны «Сохранить» и «Закрыть». При закрытии система запрашивает координаты.
6. Если визит еще не в финальном статусе, доступна «Отменить».
7. «Редактировать» открывает форму правки.
8. «Удалить» удаляет визит после подтверждения.

После закрытия или отмены редактирование становится недоступным для не-администратора.

## Привязать препарат к визиту

1. В карточке визита нажать «Редактировать» или открыть форму создания визита.
2. Прокрутить до блока «Препараты».
3. Кликнуть по полю поиска и начать вводить название.
4. Выбрать нужные препараты. Они добавятся как чипы.
5. Нажать «Сохранить» или «Создать визит».

## Посмотреть свои визиты за период

Быстрый просмотр по дням: открыть «Календарь». Вверху есть переключатель «Месяц» / «Неделя», стрелки навигации и кнопка «Сегодня». Клик по дню в режиме «Месяц» открывает список визитов этого дня, повторный клик закрывает список.

Список с фильтрами: открыть «Визиты». Там есть поиск по организации или врачу и фильтр по статусу. Для менеджера и выше доступен фильтр по сотруднику.

Сводный отчет: открыть «Отчеты». Указать период «с» и «по» или выбрать готовый пресет. При необходимости выбрать статусы. Кнопки «Excel» и «CSV» в правом верхнем углу экспортируют визиты файлом.

## Создать организацию

Доступно только администратору.

1. Открыть «Организации».
2. Нажать «Добавить» в правом верхнем углу.
3. Заполнить основную информацию, реквизиты и местоположение.
4. Нажать «Создать». Откроется карточка новой организации.

## Создать врача

Доступно только администратору.

1. Открыть «Врачи».
2. Нажать «Добавить» в правом верхнем углу.
3. Заполнить ФИО, специальность и контакты.
4. Нажать «Создать».

## Создать препарат

Доступно только администратору.

1. Открыть «Администрирование».
2. Перейти на вкладку «Препараты».
3. Нажать кнопку с плюсом.
4. Заполнить название, бренд и форму.
5. Подтвердить создание.

## Посмотреть маршрут визитов на карте

1. Открыть «Трекинг».
2. Указать период через поля даты и времени.
3. При наличии прав выбрать сотрудника.
4. На карте появятся точки закрытых визитов с координатами.

## Подсказки

- В списках «Визиты», «Организации», «Врачи» поиск работает с небольшой задержкой: нужно ввести запрос и дождаться обновления списка.
- На странице «Визиты» можно одновременно использовать поиск, фильтр по статусу и, для менеджеров и выше, фильтр по сотруднику.
- На «Карте» полезно сначала выбрать тип организации чипом «Тип», а затем искать в поле.
- Чтобы вернуться к текущей дате в календаре, нужно нажать «Сегодня».
