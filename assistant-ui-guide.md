# Навигационный гайд по Pharmo CRM

## Маршруты

### Авторизация
- `/login` — вход по логину и паролю.
- `/register` — регистрация нового аккаунта.
- `/forgot-password` — запрос ссылки для сброса пароля.
- `/reset-password` — установка нового пароля по ссылке.
- `/verify-email` — подтверждение email.

### Основное
- `/dashboard` — главная: приветствие и тепловая карта визитов.
- `/activs` — список визитов с поиском, фильтром по статусу и постранично.
- `/activs/create` — форма создания нового визита.
- `/activs/{id}` — карточка визита со статусом, временем, препаратами, действиями.
- `/activs/{id}/edit` — редактирование визита.
- `/calendar` — календарь визитов, режимы «Месяц» и «Неделя».
- `/orgs` — список организаций с поиском.
- `/orgs/create` — добавление организации (только администратор).
- `/orgs/{id}` — карточка организации.
- `/orgs/{id}/edit` — редактирование организации (только администратор).
- `/physes` — список врачей с поиском.
- `/physes/create` — добавление врача (только администратор).
- `/physes/{id}` — карточка врача.
- `/physes/{id}/edit` — редактирование врача (только администратор).
- `/map` — карта организаций по координатам.
- `/map/track` — трекинг точек закрытых визитов на карте за период.

### Аналитика и отчёты (менеджер, директор, администратор)
- `/analytics` — графики и сводки по визитам.
- `/reports` — таблица визитов за период с экспортом в CSV и Excel.

### Администрирование (только администратор)
- `/admin` — панель админа: вкладки «Пользователи», «Департаменты», «Препараты», «Специальности».
- `/admin/audit-log` — журнал изменений сущностей.

### Прочее
- `/profile` — личный кабинет: имя, фамилия, телефон, смена пароля, выход.

## Навигация

Слева — sidebar (на мобильных открывается кнопкой меню в левом верхнем углу). Sidebar разбит на группы:

- **Основное**: «Главная», «Визиты», «Календарь», «Организации», «Карта», «Трекинг», «Врачи».
- **Аналитика** (видна менеджеру, директору, администратору): «Аналитика», «Отчёты».
- **Система** (видна только администратору): «Администрирование».

Внизу sidebar — карточка пользователя с именем и ролью. Клик по ней открывает `/profile`. Там же переключатель темы и кнопка выхода. Кнопка сворачивания sidebar — у правого края панели.

В правом верхнем углу страниц-списков (Визиты, Организации, Врачи) находится кнопка действия с плюсом: «Новый визит» / «Добавить». Кнопки добавления для организаций и врачей видны только администратору.

Раздел «Препараты» отдельной страницы не имеет — он находится внутри `/admin` как одна из вкладок.

## Как выполнить типовые действия

### Создать визит
1. В sidebar открыть «Визиты».
2. Нажать кнопку «Новый визит» в правом верхнем углу.
3. В блоке «Цель визита» выбрать вкладку «Организация» или «Врач» и через поле выбрать конкретного.
4. Указать «Дата начала».
5. При желании заполнить «Описание» и в блоке «Препараты» добавить препараты через выпадающий поиск.
6. Нажать «Создать визит». Откроется карточка нового визита.

### Найти препарат
Препарат не имеет собственного списка для рядового пользователя. Найти его можно:
- в форме создания/редактирования визита, в поле «Препараты» — там работает поиск по названию;
- администратору — в `/admin`, вкладка «Препараты», поиск по названию.

### Найти врача
1. В sidebar открыть «Врачи».
2. В поле поиска ввести ФИО, специальность или контакты — список фильтруется автоматически.
3. Нажать на строку, чтобы открыть карточку врача.

### Найти организацию
1. В sidebar открыть «Организации».
2. В поле поиска ввести название, ИНН или адрес.
3. Нажать на строку для перехода в карточку организации.
4. Альтернатива — «Карта»: в правой панели карты есть тот же поиск по имени и адресу, а также фильтр по типу. Клик по элементу списка центрирует карту на организации.

### Открыть детали визита и поменять статус
1. Перейти в «Визиты» или «Календарь» и кликнуть на визит.
2. Откроется карточка с цепочкой статусов: Запланирован → Открыт → Сохранен → Закрыт. Возможен статус Отменен.
3. Снизу карточки доступны кнопки в зависимости от текущего статуса:
   - «Открыть визит» — если статус «Запланирован».
   - «Сохранить» и «Закрыть» — если статус «Открыт». При нажатии «Закрыть» система запрашивает координаты.
   - «Отменить» — если визит ещё не в финальном статусе.
   - «Редактировать» — открывает форму правки.
   - «Удалить» — удаляет визит с подтверждением.
4. После закрытия или отмены редактирование становится недоступным (для не-администратора).

### Привязать препарат к визиту
1. На карточке визита нажать «Редактировать» (или сделать это при создании).
2. Прокрутить до блока «Препараты».
3. Кликнуть по полю, в поиске начать набирать название.
4. Выбрать нужные препараты — они добавляются как чипы.
5. Нажать «Сохранить» внизу формы.

### Посмотреть свои визиты за период
- Быстрый просмотр по дням: sidebar → «Календарь». Переключатель «Месяц» / «Неделя» вверху, стрелки для навигации, кнопка «Сегодня» для возврата. Клик по дню в режиме «Месяц» открывает список визитов этого дня.
- Список с фильтрами: sidebar → «Визиты». Поиск по организации или врачу, фильтр по статусу — чипами над списком.
- Сводный отчёт за период (только менеджер и выше): sidebar → «Отчёты». Указать «с» и «по» или выбрать пресет, при необходимости — статусы. Кнопки «Excel» и «CSV» в правом верхнем углу экспортируют визиты файлом.

Менеджер, директор и администратор дополнительно видят на «Главной», «Визиты», «Календарь», «Аналитика», «Отчёты» фильтр по сотруднику.

### Создать организацию
Только администратор.
1. Sidebar → «Организации».
2. Кнопка «Добавить» в правом верхнем углу.
3. Заполнить блоки «Основная информация» (название, тип), «Реквизиты» (ИНН), «Местоположение» (адрес, координаты).
4. Нажать «Создать». Откроется карточка новой организации.

### Создать врача
Только администратор.
1. Sidebar → «Врачи».
2. Кнопка «Добавить» в правом верхнем углу.
3. Заполнить «ФИО», выбрать «Специальность», заполнить «Контакты».
4. Нажать «Создать».

### Создать препарат
Только администратор.
1. Sidebar → «Администрирование».
2. Перейти на вкладку «Препараты».
3. Нажать кнопку с плюсом, заполнить название, бренд, форму.
4. Подтвердить.

### Посмотреть маршрут визитов на карте
1. Sidebar → «Трекинг».
2. Указать период через поля даты и времени, при наличии прав — выбрать сотрудника.
3. На карте появятся точки закрытых визитов с координатами.

## Подсказки для пользователя

- В списках «Визиты», «Организации», «Врачи» поиск работает с задержкой: введите запрос и подождите — страница сбросится на первую и подгрузит результаты.
- На странице «Визиты» можно одновременно сочетать поиск, фильтр по статусу и (для менеджеров и выше) фильтр по сотруднику.
- На «Карте» полезно сначала выбрать тип организации чипом «Тип», а затем уже искать в поле — выборка сужается.
- В режиме «Месяц» календаря клик по дню раскрывает панель со списком визитов этого дня; повторный клик по тому же дню закрывает её.
- Чтобы вернуться к текущей дате в календаре, используйте кнопку «Сегодня» рядом со стрелками.
- В «Отчётах» удобнее не вводить даты вручную, а нажать готовый пресет периода, а затем при необходимости править его.
